## Descripción general

### Captura de pantalla

![](./screenshot.jpg)

Añade una captura de pantalla de tu solución. La forma más fácil es usar Firefox para ver tu proyecto, hacer clic derecho sobre la página y seleccionar "Hacer una captura de pantalla". Puedes elegir una captura de pantalla de altura completa o una recortada según la longitud de la página. Si es muy larga, puede ser mejor recortarla.

También puedes usar una herramienta como [FireShot](https://getfireshot.com/) para tomar la captura. FireShot tiene una opción gratuita, por lo que no necesitas comprarla.

Luego recorta, optimiza o edita tu imagen como desees, añádela a tu proyecto y actualiza la ruta del archivo en la imagen de arriba.
